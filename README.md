# PACT Mobile App

![PACT Logo](assets/images/pact_consultancy_pact_cover.jpg)

Welcome to the PACT Mobile App! This application is designed to provide a seamless and modern user experience for PACT Consultancy clients. Built using Flutter, the app features a sleek design, robust authentication workflows, and a consistent color scheme to reflect the PACT brand identity.

## Features

- **Authentication**: Login, Register, and Forgot Password screens with modern UI and animations.
- **Responsive Design**: Optimized for various screen sizes and orientations.
- **Custom Themes**: Centralized color and typography management for consistent branding.
- **Smooth Animations**: Enhanced user experience with subtle animations.

## Getting Started

### Prerequisites

- Flutter SDK installed ([Flutter Installation Guide](https://flutter.dev/docs/get-started/install))
- A connected device or emulator for testing

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/PACT-CONSULTANCY/PACT_mobile.git
   ```

2. Navigate to the project directory:

   ```bash
   cd PACT_mobile
   ```

3. Install dependencies:

   ```bash
   flutter pub get
   ```

### Running the App

To run the app on a connected device or emulator, use the following command:

```bash
flutter run
```

## Project Structure

- `lib/`: Contains the main application code
  - `authentication/`: Screens for login, registration, and password recovery
  - `theme/`: Centralized color and theme management
- `assets/`: Contains images and icons used in the app
